# HVAC-SYSTEM-Using-AVR-ATmega-32-
Designed and developed an intelligent HVAC control system using the ATmega32 microcontroller to optimize energy efficiency and indoor environmental quality. 
The system integrates temperature, humidity, and air quality sensing (LM35/DHT11, MQ-135) with real-time processing through ADC and implements PWM-based adaptive fan control combined with PID algorithms to ensure stable and responsive performance. 
Engineered a multi-layer architecture (sensing, control, actuation) capable of dynamically adjusting ventilation and airflow based on environmental conditions, achieving a fast response time of ~200 ms.
Incorporated fault detection mechanisms, EEPROM-based logging, and RS-485 Modbus communication for compatibility with industrial Building Management Systems (BMS). 
Simulation and testing in Proteus demonstrated reliable system behavior, reduced oscillations, and measurable energy savings of approximately 10–14% compared to conventional HVAC systems.
The results demonstrate stable system performance with reduced overshoot and oscillations due to PID control, accurate fault detection capabilities, and successful Modbus communication integration. The system achieves approximately 10–14% energy savings, making it an efficient solution for modern applications.  
It is suitable for applications in smart homes, office buildings, hospitals, industrial automation systems, and smart campuses. 
Designed as a low-cost and scalable solution, the project highlights key concepts including embedded systems, microcontroller programming, control systems, sensor integration, and industrial communication protocols. 
Overall, this work demonstrates how embedded systems combined with control engineering can be used to develop energy-efficient and intelligent infrastructure aligned with modern automation and sustainability trends.
This project highlights strong skills in embedded systems, microcontroller programming, control systems, and hardware-software integration, delivering a scalable and cost-effective solution applicable to smart buildings and industrial automation.
